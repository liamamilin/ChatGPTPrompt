2.1 NLP (自然语言处理)

2.1.1 文本生成

生成一些内容.

例如:

Prompt : 用一段话来描述天气很糟糕.

集思广益/提出想法:

ChatGPT是在工作中提出新想法的好方法。通过让它给予你关于一个特定主题的想法，你可以用它来进行你自己的个人头脑风暴会议。ChatGPT的伟大之处在于它可以访问许多人类不容易访问的信息。它读过的书、文章和文件比任何人一生所能读到的都多。这使它成为完美的助手，帮助你提出新鲜和创造性的想法。ChatGPT具有快速理解和处理大量信息的能力，可以帮助跳出思维定势，产生新的创新想法.

e.g 列出5个可持续的旅游业想法

我们可以用ChatGPT更好地进行头脑风暴的一种方法是为输出创建一个结构。在真实的生活中，我经常使用电子表格来组织我的想法。我在一栏中写下不同的想法，并在其他栏中添加关于每个想法的更多细节。这有助于我更全面地理解每个想法。我们可以用ChatGPT做同样的事情，通过以结构化的格式组织它产生的想法。这可能包括使用列表示想法的不同方面，例如潜在的好处、缺点和下一步。通过这样做，我们可以更深入地了解每一个想法，它的潜力和可行性。这可以帮助我们做出更明智的决定，并对最佳想法采取行动。

e.g 让我们集思广益，列出5个可持续的旅游业想法 。按以下格式将它们写入表格：创意名称、简短描述、可持续性如何、目标市场。

使用chatGPT改进头脑风暴的另一种方法是使用它来帮助激发创造力。一个技巧是想不起眼/不常见的想法

让我们集思广益，列出5个可持续的旅游业想法,优先考虑疯狂和古怪的想法。。

2.2 代码 Code

代码方面的应用包括:

  1. Code Generation 代码生成
  2. Code Explanation 代码解释
  3. Docstrings Generation ( 文档字符串)
  4. Programming Language Conversion (编程语言转换)
  5. Data Object Conversions (JSON, XML, CSV etc.) (数据转换) . HTML to Text (Web Scraping) (html 转换文本)

2.2.1 An prompt for Docstrings Generation

“我需要生成一个用于文档字符串的描述。这个文档字符串将用于描述一个函数或方法，以便其他开发人员能够理解其功能和如何使用它。请为一个名为[函数/方法名称]的函数（或方法）生成一个文档字符串，其中包括以下信息：1. 功能描述，2. 输入参数及其类型，3. 返回值及其类型，4. 使用示例。文档字符串应该清晰、简洁，方便其他开发人员查阅。”

2.3 结构化输出(Structured Output Styles)

  1. list Prompt: give me a list of 5 citrus fruits

  2. Numbered List

Prompt: give me a numbered list of 5 citrus fruits

  1. Headings and Subheadings

Prompt: convert this text into headings and subheadings: Babe Ruth joined the New York Yankees in 1920. The Boston Red Sox sold his contract to the Yankees. He played for the Yankees from 1920 to 1934, and during that time he established himself as one of the greatest players in baseball history.

  1. Tables

Prompt: create a table from this list: Oranges, Lemons, Limes, Grapefruit, Tangerines

2.4 非结构化输出(Unstructured Output Styles)

  1. Narrative Modes (1st, 2nd or in the 3rd person)

Prompt: write a paragraph on how to make brownies in the 1st person

  1. Formal Prompt: write a paragraph on the topic of cellular automata in a formal style

  2. Informal

Prompt: write a paragraph on the topic of cellular automata in an informal style

  1. Personas

Prompt: write a paragraph on the topic of cellular automata in the style of a social media influencer

  1. Custom Text Manipulation

Prompt: write a paragraph on the history of the calculator, include emojis at the end of every sentence, and do not capitalize the first word in each sentence
